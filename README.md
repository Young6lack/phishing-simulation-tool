# 🎣 Phishing Awareness Simulation Tool

**Live Demo**: [Try the App](https://your-link.streamlit.app)  
**GitHub**: [View Source Code](https://github.com/Young6lack/phishing-simulation-tool)

A beginner cybersecurity project that demonstrates real-world phishing techniques in a safe educational environment.

---

## ✨ Features
- 📧 **Send Real Phishing Emails** (using Gmail)
- 🌐 **Realistic Fake Login Page** (Credential harvesting demo)
- 📊 **Live Attack Dashboard** (Shows captured credentials)
- Educational explanations at every step

---

## 🛠️ Technologies Used
- Python 3
- Streamlit
- Pandas
- SMTP Email (Gmail)

---

## 🚀 How to Run Locally

```bash
git clone https://github.com/Young6lack/phishing-simulation-tool.git

python3 -m venv venv
